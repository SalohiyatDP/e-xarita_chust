# Chust District Electronic Land Balance Mapping System

Goal:

Automatically generate electronic land balance maps for all massifs of Chust district.

Input:

- Personal Geodatabase (.mdb)
- ArcMap Document (.mxd)
- Layer file (.lyr)
- Land balance Excel (.xls)

Output:

- PDF electronic map
- PDF land balance report
- JPG preview
- District summary report

Reference:

A.Navoiy massif sample map and table.
